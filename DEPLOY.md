# CRSCWP 플랫폼 실행 가이드

본 플랫폼은 `README.md`의 요구사항에 따라 구현된 로봇 / 스포츠 / 커피 동호회 웹 플랫폼입니다.

## 1. 로컬 실행 방법

1. **가상 환경 활성화:**
   ```powershell
   .\venv\Scripts\activate
   ```

2. **애플리케이션 실행:**
   ```powershell
   python run.py
   ```

3. **접속:**
   브라우저에서 `http://127.0.0.1:5000`으로 접속합니다.

## 2. 주요 기능
- **회원가입 및 로그인:** 중복 가입 방지 및 2단계 인증(2FA) 시뮬레이션 기능이 포함되어 있습니다. (인증 코드는 임의의 6자리 숫자 입력)
- **분야별 게시판:** AGI, ACTF, 수영, 스카이다이빙, 암벽등반, 카페 취업, 기술 연마의 7개 카테고리를 지원합니다.
- **문서 작성:** 로그인한 회원은 각 분야별로 자유롭게 문서를 작성하고 공유할 수 있습니다.
- **프로필:** 본인이 작성한 문서 목록을 확인할 수 있습니다.

## 3. 배포 안내
본 플랫폼을 실제 서버에 배포하려면 다음 서비스를 권장합니다:
- **Render.com / Fly.io:** Flask 앱 배포에 적합한 서비스입니다.
- **PythonAnywhere:** 초보자가 사용하기 쉬운 Python 전용 호스팅 서비스입니다.

배포 시 `run.py` 대신 `gunicorn` 등의 WSGI 서버를 사용하고, `SECRET_KEY`를 안전하게 관리하십시오.
